🚀 ÚLTIMO PASSO - CONFIGURAR NETLIFY (2 minutos)
════════════════════════════════════════════════════════════════

1️⃣  ACESSAR NETLIFY
   ─────────────────────────────────────────────────────────

   Abra no navegador:
   
   👉 https://app.netlify.com
   
   Faça login com:
   - Conta GitHub (recomendado - mais fácil!)
   - OU crie uma conta nova


2️⃣  IMPORTAR PROJETO DO GITHUB
   ─────────────────────────────────────────────────────────

   No dashboard do Netlify:
   
   1. Clique no botão: [Add new site]
   
   2. Escolha: [Import an existing project]
   
   3. Clique em: [Deploy with GitHub]
   
   4. Se pedir, autorize o Netlify a acessar seus repositórios
   
   5. Procure e selecione:
      📁 ivoneieletrotecnico-collab/151230


3️⃣  CONFIGURAR O DEPLOY
   ─────────────────────────────────────────────────────────

   Na tela de configuração:
   
   Branch to deploy:
   main
   
   Base directory:
   (deixe vazio)
   
   Build command:
   (deixe vazio)
   
   Publish directory:
   .
   
   Clique em: [Deploy ivonei-eletrotecnico]


4️⃣  AGUARDAR O DEPLOY (30-60 segundos)
   ─────────────────────────────────────────────────────────

   O Netlify vai:
   ✓ Conectar ao GitHub
   ✓ Baixar os arquivos
   ✓ Processar o site
   ✓ Publicar online
   
   Você verá o progresso na tela!


5️⃣  ACESSAR SEU SITE
   ─────────────────────────────────────────────────────────

   Após deploy completo, o Netlify mostrará:
   
   URL temporária (exemplo):
   https://wonderful-curie-a1b2c3.netlify.app
   
   
   PERSONALIZAR A URL:
   
   1. No dashboard, clique em: [Site settings]
   
   2. Vá em: [Domain management]
   
   3. Clique em: [Options] > [Change site name]
   
   4. Digite: ivonei-eletrotecnico
   
   5. Clique em: [Save]
   
   
   URL FINAL:
   👉 https://eletrotecnico.vercel.app
