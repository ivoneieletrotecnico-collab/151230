## 🚀 Deploy Automático

Este projeto está configurado com deploy automático via **GitHub + Netlify**.

Toda alteração enviada para o repositório GitHub é automaticamente publicada.

### Como fazer alterações:

```bash
# 1. Editar o arquivo index-static.html
# 2. Fazer commit
git add .
git commit -m "Descrição da alteração"

# 3. Enviar para GitHub (dispara deploy automático)
git push
```

**Deploy acontece em segundos!** ⚡
