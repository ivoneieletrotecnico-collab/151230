📝 PERSONALIZAR OS DOWNLOADS
════════════════════════════════════════════════════════════════

OPÇÃO 1: Através do Painel Admin
1. Acesse: https://ivonei-eletrotecnico.netlify.app/admin-login.html
2. Faça login
3. Adicione seus próprios downloads
4. Os dados sobrescrevem os exemplos

OPÇÃO 2: Editar o arquivo downloads-data.js
1. Abra: downloads-data.js
2. Edite o array DOWNLOADS_DATABASE
3. Altere: nome, descrição, URL, tamanho, etc.
4. Salve e faça deploy
