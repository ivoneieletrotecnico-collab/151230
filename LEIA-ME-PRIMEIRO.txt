🎯 PRÓXIMOS PASSOS
════════════════════════════════════════════════════════════════

Se você quer DEPLOY AUTOMÁTICO:

1. Instale o Git (se não tiver)
   https://git-scm.com/download/win

2. Execute: setup-git.ps1
   (clique direito > Executar com PowerShell)

3. Crie repositório no GitHub
   https://github.com/new
   Nome: ivonei-eletrotecnico

4. Conecte com Netlify
   https://app.netlify.com
   Add new site > Import from GitHub

5. PRONTO! Cada alteração será publicada automaticamente ⚡


💡 DICA RÁPIDA
════════════════════════════════════════════════════════════════

Para fazer alterações no site:
• Edite o arquivo: index-static.html
• Procure por textos, cores, imagens, etc.
• Salve o arquivo
• Se tiver GitHub configurado: git add . && git commit -m "atualização" && git push
• Se não: faça upload manual no Netlify
