# COMANDOS GIT E DEPLOY - VERCEL

## Publicar alteracoes

```bash
git status
git add .
git commit -m "Atualizacao do site"
git push origin master
```

## Ver historico

```bash
git log --oneline --decorate -n 10
```

## Ver diferencas locais

```bash
git diff
```

## Se o site nao atualizar

1. Confirme que o `git push` terminou sem erro.
2. Abra `https://vercel.com/dashboard`
3. Verifique a aba `Deployments`
4. Se necessario, rode um novo push com outro commit
